O HTML (HyperText Markup Language) é a linguagem de marcação padrão utilizada para criar e estruturar páginas na web. Ele serve como a base fundamental de praticamente todos os sites existentes na internet. Diferente de linguagens de programação tradicionais, o HTML não executa cálculos ou cria funcionalidades complexas por si só; sua principal função é organizar e estruturar o conteúdo, definindo o que é um título, um parágrafo, uma imagem, um link, uma tabela ou um formulário.

O funcionamento do HTML é baseado em tags, que são elementos delimitados por sinais de menor e maior (<>). Essas tags indicam ao navegador como o conteúdo deve ser interpretado e exibido. A maioria das tags possui uma abertura e um fechamento, como p, e /p, que definem o início e o fim de um elemento. Existem também elementos que podem conter atributos, responsáveis por fornecer informações adicionais, como endereços de links, identificadores ou classes para estilização.

O HTML normalmente trabalha em conjunto com outras tecnologias essenciais da web. O CSS (Cascading Style Sheets) é responsável pela parte visual, como cores, fontes, espaçamentos e layout. Já o JavaScript adiciona interatividade e dinamismo às páginas, permitindo criar animações, validar formulários e atualizar conteúdos sem recarregar a página.
