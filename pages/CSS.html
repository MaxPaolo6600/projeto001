O CSS (Cascading Style Sheets) é a linguagem utilizada para definir a aparência e o estilo de páginas desenvolvidas com HTML. Enquanto o HTML é responsável por estruturar o conteúdo de um site, o CSS cuida da parte visual, determinando como os elementos serão exibidos na tela. Isso inclui cores, fontes, tamanhos, espaçamentos, posicionamento, animações e até mesmo o layout completo da página.

O funcionamento do CSS baseia-se na aplicação de regras de estilo aos elementos HTML. Cada regra é composta por um seletor e um bloco de declarações. O seletor indica qual elemento será estilizado, e as declarações definem as propriedades e seus respectivos valores. Por exemplo, é possível alterar a cor de todos os parágrafos de uma página, definir o tamanho de um título ou centralizar um conteúdo específico.

Uma das principais características do CSS é o conceito de “cascata”, que determina como as regras são aplicadas quando há conflitos entre estilos diferentes. Essa lógica considera fatores como a especificidade do seletor, a ordem das declarações e a importância atribuída a determinadas regras. Isso permite um controle detalhado sobre a aparência dos elementos, tornando possível personalizar cada parte do site com precisão.
